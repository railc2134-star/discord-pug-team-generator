# Discord Mini Game Bot

A bot to manage mini games on Discord. It allows users to start a new game and join existing ones. The bot uses Discord.py library and supports multiple game modes.

## Features:
- Start a new game with /start command
- Join an existing game with /join command
- Supports multiple game modes (1v1, 2v2, 3v3, 4v4, 5v5)

## Technologies used:
- Discord.py library
- Python 3

## How to run it:
1. Install the required libraries by running pip install -r requirements.txt
2. Replace your_discord_token_here with your actual Discord bot token
3. Invite the bot to your Discord server
4. Use the /start command to start a new game
